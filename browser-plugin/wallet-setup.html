NINK local wallet setup

One-time MetaMask setup for on-chain sign-off on your local Hardhat chain. These are public test keys — never use them on mainnet.

Step 1 — Install MetaMask

  1. Install the MetaMask browser extension if you do not have it yet.
  2. Create a new wallet or import an existing one — any MetaMask profile works.
  3. Unlock MetaMask before NINK sign-off.
Install MetaMask

Step 2 — Add the local Hardhat network

MetaMask → network dropdown → Add network → Add a network manually

Step 3 — Import a funded test account

MetaMask → account icon → Import account → paste private key below.

This account starts with 10,000 test ETH on every fresh Hardhat node.

Step 5 — Sign off in NINK

  1. Reload the NINK extension at chrome://extensions.
  2. Turn Local test mode OFF in the NINK popup.
  3. Refresh your AI chat tab (ChatGPT, Claude, etc.).
  4. Click Press NINK to Sign-Off — confirm in MetaMask on the chat tab.
